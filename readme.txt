Easily add google Analytic code from the wordPress administration screen.

== Description ==

This is a simple plugin which allows you to include your Google Analytics tracking code in the website.

== Installation ==

1. Upload `sgabst-simple-google-analytics` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Option will be visible in the `setting` menu i.e. Add Google Analytics option.

NOTE: 1] Tracking code will not be active for the logged-in admin user.*
      2] Please add valid tracking code in the text box of the plugin setting page.
      3] If you will put invalid code or any other value instead of Google analytic code, then analytic code visible in the head section with invalid code.
